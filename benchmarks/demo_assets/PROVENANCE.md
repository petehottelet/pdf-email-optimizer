# Demo asset provenance

All images in this directory are **synthetic, AI-generated stock** — there are no
real people, properties, brands, or trademarks in them, so they are safe to
redistribute. They were generated in April 2026 purely to demonstrate the
optimizer on realistic, photo-heavy layouts.

Each demo brochure is assembled from the curated set below by
[`benchmarks/make_demo_brochures.py`](../make_demo_brochures.py). The committed
files are re-encoded to optimized JPEG to keep the repository small; the
*demonstration* PDFs embed them losslessly to mimic a high-quality design-tool
export (Illustrator / InDesign / Canva / Keynote). The descriptions below were
written from the images themselves.
