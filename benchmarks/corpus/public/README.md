# Public benchmark fixtures

This directory is where redistributable real-world PDFs go: CC0, public domain, your own generated material, or PDFs whose owner has explicitly authorized redistribution under the project's MIT license.

The reproducible public corpus today is the synthetic, CC0 set in [`benchmarks/fixtures/`](../../fixtures/), generated by [`benchmarks/make_fixtures.py`](../../make_fixtures.py). Anything dropped under `public/<id>/` should follow the same layout:

```
public/<id>/
  input.pdf
  notes.md
```

`notes.md` documents `id`, `category`, `description`, `license`, `source`, and `expected_behavior` (see [`../README.md`](../README.md) for the required fields).

Submit external fixtures via the [fixture submission template](../../../.github/ISSUE_TEMPLATE/fixture-submission.yml).
